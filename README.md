# finalMicros
Proyecto final de microcomputadoras

Consiste en un sistema de vigilancia. Se puede controlar a través de Bluetooth, implementa un modo de búsqueda de oponente y reaccción a movimientos.

# Componentes usados
1. Raspberry Pi Pico
2. Sensor RWCL 9610A
3. HC-SR04
4. Display LCD
5. Display 7 segmentos
6. Módulo HC-05
7. Push button
8. Potenciómetro
9. Servomotores SG90
10. Buzzer
11. Acelerómetro MPU 6050
12. Láser
